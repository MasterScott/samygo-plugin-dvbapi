Introduction

The plugin allows connect Samsung TV to OScam. It creates a software CAM
and communicates with OSCam over camd.socket (using 'dvbapi' module).
This way it pass decryption requests to OSCam and obtains back keys which
are needed for CSA stream decryption.

Tutorial

This version of the plugin allows to connect to any OSCam installation via TCP socket

OSCam dvbapi Settings

[dvbapi]
enabled                       = 1
au                            = 1
pmt_mode                      = 4
listen_port                   = 20000
user                          = localuser
boxtype                       = pc

Command line arguments

  • OSCAM_SERVER_IP:xxx.xxx.xxx.xxx - OSCAM dvbapi server address
  • OSCAM_SERVER_PORT:xxxxx - OSCAM dvbapi listenport

Example call format

samyGOso -D -r -l /mtd_rwdata/oscam/libdvbapi.so OSCAM_SERVER_IP:192.168.1.48 OSCAM_SERVER_PORT:20000
